## 什么是 MCP？

MCP (Model Context Protocol) 是一个标准化协议，使 AI 助手能够访问外部数据源和工具。@movk/core 提供 MCP 服务器，让 AI 工具能够直接访问函数信息、源代码和使用示例。

## 可用资源

@movk/core MCP 服务器提供以下资源供 AI 发现和浏览：

| 资源 URI | 描述 |
|---------|------|
| `resource://movk-core/all-functions`{lang="ts-type"} | 所有可用函数的完整列表，包含元数据和分类信息 |
| `resource://movk-core/tree-operations`{lang="ts-type"} | 树形结构操作的专题文档（核心特性） |

在 Claude Code 等工具中使用 `@` 访问这些资源。

## 可用工具

| 工具名称 | 描述 |
|---------|------|
| `list-functions`{lang="ts-type"} | 列出所有 @movk/core 函数及其分类和基本信息 |
| `get-function`{lang="ts-type"} | 获取特定函数的详细文档与使用示例，需提供 `functionName`（camelCase 或 kebab-case） |
| `search-functions`{lang="ts-type"} | 按 `category`、`subcategory` 或 `keyword` 搜索函数 |

## 可用提示

@movk/core MCP 服务器为常见工作流提供引导式提示：

| 提示名称 | 描述 |
|---------|------|
| `find-function-for-usecase`{lang="ts-type"} | 根据使用场景推荐最适合的函数 |

在 Claude Code 等工具中使用 `/` 访问这些提示。

## 配置

@movk/core MCP 服务器使用 HTTP 传输，可在不同 AI 助手中配置。

### Claude Code

::note
确保已安装 Claude Code。参阅 [Anthropic 文档](https://docs.anthropic.com/en/docs/claude-code/quickstart) 获取安装说明。
::

通过 CLI 命令添加服务器：

```bash
claude mcp add --transport http movk-core https://core.mhaibaraai.cn/mcp
```

### Claude Desktop

1. 打开 Claude Desktop，进入「Settings」>「Developer」
2. 点击「Edit Config」打开本地 Claude 配置目录
3. 编辑 `claude_desktop_config.json`：

```json [claude_desktop_config.json]
{
  "mcpServers": {
    "movk-core": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://core.mhaibaraai.cn/mcp"
      ]
    }
  }
}
```

4. 重启 Claude Desktop，@movk/core MCP 服务器即注册成功

## 使用示例

配置完成后，您可以向 AI 助手提出如下问题：

**发现与查询**
- 「列出所有可用的 @movk/core 函数」
- 「获取 `chunk` 函数的文档」
- 「`Tree.fromList` 接受哪些参数？」
- 「查找数组相关的函数」

**树形结构（核心特性）**
- 「列出所有树形结构操作函数」
- 「我需要将树形数据转为扁平列表，用什么函数？」
- 「如何按字段过滤树并保留祖先节点？」

**按场景选型**
- 「字符串要转成 kebab-case，用哪个函数？」
- 「如何深度合并多个配置对象？」
- 「展示所有 URL 查询参数处理函数」
- 「获取 `useAppStorage` 的使用示例」

AI 助手会通过 MCP 工具获取结构化数据并给出场景化建议；带「推荐 / 应该用什么」的提问会自动触发引导式 prompts。
