## 什么是 Agent Skills？

Agent Skills 是按 [Agent Skills 规范](https://agentskills.io/specification) 编写的 `SKILL.md` 结构化知识文件。与 [MCP Server](/docs/getting-started/ai/mcp)（提供实时文档检索工具）不同，Skills 直接加载进 AI 编码代理的上下文，让代理在整段对话中持续参考。

@movk/core 提供单一 `movk-core` 技能，覆盖校验、数组、异步、URL、字符串、对象、树形、文件、路径与 Vue 组合式等能力域，注入「在何种场景使用哪个函数、如何正确书写」的指引，让生成的代码符合本库约定。

## 使用

`movk-core` 技能让 AI 代理掌握使用 @movk/core 的完整知识：

- **校验（validators）**：`isArray`、`isObject`、`isPlainObject`、`isEmpty` 等类型守卫
- **工具（utilities）**：数组 `chunk`/`unique`、异步 `debounce`/`throttle`、URL 解析与查询参数、CSS `lengthToPx`
- **转换（transformers）**：命名风格转换、对象键转换、`Tree` 树形结构操作、`stringifyMinimark`
- **辅助（helpers）**：`deepClone`/`deepMerge`、`pick`/`omit`/`separate`、`equalsBy`、嵌套路径读写、文件处理
- **Composables**：`useAppStorage`、`useCopyCode`、`useInfiniteScrollBinding`、`useOverflowDetection`

技能入口 `SKILL.md` 承载跨域铁律（按需导入、纯函数不可变、不臆造 API、Tree config）与模块/用例路由表，让代理在编写代码前先选对函数。

::note
安装后，可在 Agent 聊天中输入 `/movk-core` 触发技能。
::

### Skills CLI

[`skills`](https://skills.sh) CLI 是安装 @movk/core 技能的最便捷方式，支持 35+ 代理，包括 Cursor、Claude Code、Codex、Windsurf、Cline 等。直接传入文档站地址即可，CLI 会自动抓取技能：

```bash
npx skills add https://core.mhaibaraai.cn
```

通过 `--agent` 指定目标代理：

```bash
npx skills add https://core.mhaibaraai.cn --agent cursor
npx skills add https://core.mhaibaraai.cn --agent claude-code
```

或使用 `--global` 全局安装，让技能在所有项目中可用：

```bash
npx skills add https://core.mhaibaraai.cn --global
```

::tip
访问 [skills.sh](https://skills.sh) 了解 Skills 生态并浏览更多可用技能。
::

### Cursor

**一键安装**：点击下方按钮，直接在 Cursor 中安装 @movk/core 技能。

::u-button
---
to: "cursor://anysphere.cursor-deeplink/install-skill?url=https://github.com/mhaibaraai/movk-core/tree/main/docs/skills/movk-core"
label: 安装 Skill
color: neutral
icon: i-simple-icons-cursor
---
::

**手动配置**：进入「Settings」>「Skills」，点击「Add skill」，粘贴以下 URL：

```text
https://github.com/mhaibaraai/movk-core/tree/main/docs/skills/movk-core
```

### Claude Code

::note
确保已安装 Claude Code。参阅 [Anthropic 文档](https://docs.anthropic.com/en/docs/claude-code/quickstart) 获取安装说明。
::

通过 CLI 命令添加技能：

```bash
claude skill add https://github.com/mhaibaraai/movk-core/tree/main/docs/skills/movk-core
```
